# v0.1.2 更新日志

发布日期：2026-02-23
范围：`v0.1.1...v0.1.2`
基准提交：`ed84f9e`

## 重点更新

- 联机大厅支持从公共 CDN 动态获取节点列表，提升联机成功率。
- 修复下载版本解压时版本号被误判为扩展名导致截断的问题。

## 新增

- **联机大厅（EasyTier）**
  - 支持从 `https://et-public-node.roundstudio.top/` 动态获取公共节点列表
  - 使用 Cloudflare CDN 优选 IP 加速节点列表获取，降低连接延迟
  - 添加节点列表缓存机制（TTL 10 分钟），减少 CDN 请求频率
  - 更新默认引导节点为 `tcp://public.easytier.bmcbl.com:54321`

## 修复

- **下载与解压**
  - 修复版本号（如 `1.18.31`）被误判为文件扩展名导致截断成 `1.18` 的问题
  - 新增 `strip_known_archive_extension` 函数，仅移除已知的归档扩展名（`.msixvc`/`.appxbundle`/`.appx`/`.zip`）
  - 优化下载后解压的文件名处理逻辑，优先使用实际下载文件名

## 变更

- **本地化**
  - 更新所有语言文件的引导节点占位符文本

## 升级提示

- 联机大厅节点列表将自动从 CDN 获取，无需手动配置引导节点
- 如遇到节点获取失败，将自动回退到内置的默认节点
