# v0.1.1 更新日志

发布日期：2026-02-21  
范围：`v0.0.7...v0.1.1`  
基准提交：`1030cb6`

## 重点更新

- 重构大部分前端页面与交互，整体体验与稳定性提升。
- 支持 GDK 版本游戏下载、安装与启动流程。
- 调整注入方式，增强对第三方加载器/注入方案的兼容性。
- 联机大厅（EasyTier）在稳定性与安全策略（ACL）方面进行了大量增强，并内置 `wintun.dll` 以提升创建虚拟网卡的成功率。

## 新增

- **GDK 支持**
  - 增加 GDK 依赖与相关流程，支持 GDK 游戏包下载安装与启动。
- **联机大厅（EasyTier / PaperConnect）**
  - 新增 NAT 检测能力与相关 UI/逻辑。
  - 支持多端口/自动选择协议端口、握手与心跳重试等稳定性改进。
  - 引入/完善应用层 ACL 规则，限制不必要的对等流量，降低安全风险。
  - 内置并自动加载 `wintun.dll`（构建期嵌入，运行期按需落盘并 `LoadLibrary` 预加载），方便创建虚拟网卡。
- **版本级设置**
  - 增加按版本维度的注入、鼠标锁定、编辑器模式等配置项与 UI。
- **导入与下载体验**
  - 增加资源导入弹窗与解压性能优化。
  - 下载逻辑增强：多线程下载优化、支持自定义 UA、代理/镜像相关改进等。
- **UI/可用性**
  - 全局字体与若干组件样式优化（含提示/tooltip 体系、关于页赞助者弹窗等）。

## 变更

- **配置文件版本化**
  - 在 `settings.toml` 引入 `config_version`，用于后续迁移与兼容更新。
  - 对旧配置进行自动迁移；当检测到旧配置（无 `config_version`）时，会重置 `agreement_accepted` 以触发更新后的协议确认。
- **用户协议**
  - 更新“隐私与数据安全”与“联机协议”条款，并对协议结构进行了规范化整理（更清晰的定义、承诺、免责声明与争议解决条款）。
- **构建/工程**
  - 引入 conventional commits 约束与相关 CI/脚本调整。
  - EasyTier 依赖与构建配置更新（包含 fork/运行时依赖拷贝逻辑等）。

## 修复

- **CurseForge 页面**
  - 分页布局与暗色主题样式修复。
  - “分享导入/分页”相关文案本地化补齐，修复分页信息占位符传参不一致导致的回退显示问题。
- **启动与设置**
  - 版本下拉等 UI 动画/稳定性问题修复。
  - 更新检查、进度条与下载速度统计等细节问题修复。
- **联机稳定性**
  - 解决重启绑定失败、端口/白名单策略、发现广播等相关问题，提升联机成功率与一致性。

## 升级提示

- 若用户从 `v0.0.7` 升级到 `v0.1.1`：
  - 配置会自动迁移并写入 `config_version`；旧配置会触发协议重新确认。
  - 如不希望上传基础运行信息，可在 `设置 -> 启动器` 中关闭相关选项。
