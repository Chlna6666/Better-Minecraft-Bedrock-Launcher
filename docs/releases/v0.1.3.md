# v0.1.3 更新日志

发布日期：2026-02-25
范围：`v0.1.2...v0.1.3`
基准提交：`3fba53e`

## 重点更新

- 导入解析兼容 `manifest.json` 的 `format_version: 3` 常见写法，降低“找不到 manifest”的误报。
- 修复 CurseForge 列表/网格视图切换在暗色主题下的样式问题。

## 修复

- **导入**
  - 支持 `header.version` / `modules[].version` 使用字符串或数组两种写法
  - `manifest.json` 文件名大小写不敏感匹配（兼容 `Manifest.json` 等）
  - 解析前移除 UTF-8 BOM，避免 JSON 解析失败
- **CurseForge**
  - 补齐暗色主题下列表/网格切换按钮的背景/文字/分隔线样式
