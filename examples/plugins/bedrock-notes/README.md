# BMCBL Bedrock Notes

BMCBL Bedrock Notes shows recent Minecraft Bedrock patch notes directly on the BMCBL home page.

The plugin does not receive direct network access. BMCBL fetches the Mojang patch notes endpoint through a host-managed HTTPS allowlist cache and gives the plugin a cached text snapshot.

Data source:

`https://launchercontent.mojang.com/v2/bedrockPatchNotes.json`
